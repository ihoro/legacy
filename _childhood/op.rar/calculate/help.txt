Private Calculator v1.06
Калькулятор для расчета сумм от сложных процентов, когда установленный процент от исходной суммы   прибавляется к исходной сумме с каждым циклом. Точность расчета - 8 знаков после запятой.
- Start Summ - исходная сумма;
- Percent - установленный процент;
- N - количество циклов;
- Result - результат.
Расчет производится для любых переменных перечисленных выше. Для этого в окна вводятся все числа кроме искомого. Результат выводится щелчком по кнопке "Calculate" или нажатии "Enter" на клавиатуре. После чего исходные данные можно изменять и получать искомое число в том же поле. Для смены искомого числа его поле нужно оставить пустым. Для этого можно использовать кнопки "Х" соответственно для каждого поля.
Щелчком по стрелке, число из окна "Result" передается в окно "Start Summ", "Result" очищается.
