OIVSoft Quake3 Demo Viewer v.1.0

Используется для просмотра демок игры quake3(osp).

Как юзать:
  К примеру, имеется дема под именем 'some-demo.dm_68'. Пакуем ее rar'ом (zip,...,все что, поддерживает winrar v3.x), для удобства делаем ринейм архива в '*.q3d' (q3d = quake3 demo). Удобство заключается в том, что это расширение прописано за программой, и вам достаточно будет дабл-клика для просмотра демки. Довольно просто...
  Для настроек используется программа q3dvs.exe, с помощью которой нужно указать пути к таким файлам: quake3.exe, rar.exe, some.cfg. Последний файл - конфигурация, в которой хотелось бы просмотреть демку.
  Вывод: демки всумме занимают меньше места, особенно, когда их много (зачастую так и есть). Поскольку эта информация нечасто используемая, поэтому целесообразней было бы хранить ее в запакованном виде.


P.S. В след. версии предпологается поддержка других модов (тот же defrag).
